# Legacy GIMP 2 plug-in (unsupported)

These files are inherited from the upstream
[Shriinivas/gimpsegany](https://github.com/Shriinivas/gimpsegany) project and
target **GIMP 2.10** (Python 2 / Python-Fu).

This fork focuses exclusively on **GIMP 3** (see `seganyplugin.py` /
`seganybridge.py` in the repo root). The files here are kept for reference only:

- `seganyplugin_GIMP2.py`
- `seganybridge_GIMP2_SAM1.py`

They are **not maintained, tested, or covered by CI**, and the macOS / Apple
Silicon / SAM 2.1 improvements of this fork are not applied to them.
